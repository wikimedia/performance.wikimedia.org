We monitor the performance of Wikipedia continuously. These graphs are moving medians of measurements from 1 in 1000 sampled requests. If you want to dig deeper into our metrics, check out our Grafana graphs.

– Wikimedia Performance Team

Metrics

domInteractive: The browser has parsed the main HTML document.

firstPaint: When something is first painted on-screen.

loadEventEnd: The load event fired in the browser and the document is complete.

responseStart: Time to first byte on the client from the server.

saveTiming: Time to submit and save an edit on an article.
